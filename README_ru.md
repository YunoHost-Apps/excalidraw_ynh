## Обзор

Excalidraw is a virtual collaborative whiteboard tool that lets you easily sketch diagrams that have a hand-drawn feel to them.


**Поставляемая версия:** 0.17.3~ynh2

**Демо-версия:** <https://excalidraw.com>
